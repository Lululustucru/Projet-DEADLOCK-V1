# DEADLOCK ![logo](./images/a03c87f5-8ba2-4b22-8ff0-f274fae28a25.jpg)
Jeu d’énigmes inspiré d'Unlock!
> Projet de 1ère année – Cybersécurité – IPSSI - 2025/2026  
> Équipe : Ilona Delmotte, Cassini-Peloux Luca, Ayoub, Dorian Feraud, Mathis, Sevan, Maël Costa

# Objectif du projet
- Créer une jouable en solo et en ligne.  
- Mélanger logique, collaboration et cybersécurité  
- Découvrir le développement web complet (front + back + sécu). 
- Apprendre le travail d’équipe et la gestion de projet.

# Concept du jeu
- Le joueur résout des énigmes et casse-têtes pour progresser.  
- Chaque scénario est une suite de cartes interactives.  
- Objectif : Réussir à compléter l'objectif avant la fin du chrono.  
- Mode solo ou coopératif en ligne  
- Les indices et les pénalités disponibles pour équilibrer la difficulté.

# Technologies
- Front-end (Ayoub, Romain, Maël Costa, Cassini-Peloux Luca) : HTML, CSS et JavaScript <=> React + Vite.
- Back-end (Sevan, Mathis, Dorian Feraud, Cassini-Peloux Luca) : Javascript, Python, PHP, C# <=> Firebase.
- UI/Design (Ilona Delmote, Cassini-Peloux Luca) : HTML, Javascript, CSS <=> TailwindCSS, Figma, IA génératives (Kling, Vozart, Mureka)
- Hébergement : Vercel
- Collaboration : Github, Discord
- Tests : ESLint, Prettier, Playwright
